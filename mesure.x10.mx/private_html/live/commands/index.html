Stream Commands

When Mesure is live, you can send commands. This is a list of all the commands that you have.

Commands Available for Everyone

!help - Sends the link to Mesure's stream commands list.

!faq - Sends the link to Mesure's stream FAQ.

!discord - Sends the link to Mesure's Discord server.

Moderator Only Commands

!title (title) - Sets the title of the stream.

!msg (message) - Displays a message as text on the stream.
