Why I like web development so much

If you did not know, I made this website and everything on it myself. Yes, I did take some stuff from the internet, but everything you see was configured by me, especially the design. I enjoy web development a lot because it has so much creative freedom involved with it, and there is no correct or incorrect way to do it.

What is web development?

Web development is making websites using HTML, JS, and CSS. There are also other things, like PHP and mySQL, for databases, and ReactJS, Vue, or jQuery to make it easier. My website only uses HTML, JS, and CSS, mostly because I am too lazy to add a framework. (Also because I hate you, Minystreem/ElectricTurtle /j)

Essentially, web development is just developing a website, even if it isn't hosted or just on a local file (e.x. file://(file path) on most web browsers).

Why I think web development is fun

I think web development is fun because there is a lot of creative freedom, and I can basically do whatever I want. Every new website that I make, I learn something new about web development. This website (probably the main one I am going to use for at least a few years) has taught me a lot about making websites, including new things in CSS that I previously did not know.

Web development for me is mostly just a learning opportunity - I can make whatever I want without limitations, and if I can't figure something out, W3Schools and Stack Overflow can help with whatever problem I'm having.
